What is unordered list in HTML?

An Unordered list is used to create a list of related items, in bulleted or unordered format. It starts with the ul tag, followed by the li tag to show list items inside ul tag.

What is ordered list in HTML?

The Ordered lists have an order which is either numerical or alphabetical. The ol tag is used to create ordered lists in HTML and just like unordered list, we use li tag to define or show lists inside ol tag.

What is description list in HTML?

JA description list is a type of list where each item has a description. It is also known as a definition list. The dl tag is used to create description list, the dt tag defines the item, and the dd tag describes each item in list.
